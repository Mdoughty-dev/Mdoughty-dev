## 🔬 What I’m Exploring Now

### 🧠 OpenFace + Surreal Art

I’m currently experimenting with **OpenFace 3.0** to see if facial data can be used in more abstract, creative ways.

The idea is not just detection (emotions, landmarks, etc.), but:
- Interpreting subtle signals
- Mapping them into generative or surreal outputs
- Exploring whether systems can reflect something closer to a “subconscious” state

It’s early and messy — but that’s the point.
